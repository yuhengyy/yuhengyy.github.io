# 三消游戏（Dotween版）

## 项目介绍
在「基础版三消游戏」的基础上，用Dotween插件优化动画效果，提升视觉体验，核心优化点：
1. 方块拖拽：增加平滑移动动画（替代原生Transform移动）
2. 消除效果：增加爆炸、碎片消散动画
3. 分数弹出：分数文本向上漂浮+渐隐
4. 关卡切换：场景淡入淡出过渡

## 关键技术
- Dotween插件的基础使用（DOJump、DOFade、DOMove）
- 动画序列（Sequence）控制多步动画
- 动画事件回调（如动画结束后执行补位逻辑）

## 插件依赖
- Dotween Free（可从Unity Asset Store下载）
