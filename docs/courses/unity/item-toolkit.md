# 物品工具包

## 项目介绍
通用的游戏物品管理工具包，包含完整的物品生命周期管理：
1. 物品定义：支持武器、道具、消耗品等类型，可配置属性（攻击力、使用次数等）
2. 背包系统：格子式背包，支持物品拖拽、叠加、丢弃
3. 物品使用：消耗品恢复血量/蓝量，武器装备后生效属性
4. 物品获取：通过任务奖励、商店购买、怪物掉落触发

## 关键技术
- 工厂模式创建不同类型物品
- 事件系统通知物品状态变化（如背包满了、物品使用成功）
- UI格子与物品数据绑定

## 复用性
可直接导入其他Unity项目，只需修改物品配置表即可适配不同游戏。
