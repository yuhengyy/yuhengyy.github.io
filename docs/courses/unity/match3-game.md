# 三消游戏（基础版）

## 项目介绍
基于Unity实现的基础三消游戏，核心功能包括：
1. 方块生成与布局（5x5网格）
2. 鼠标拖拽交换方块
3. 匹配判定（横向/纵向3个及以上相同方块）
4. 消除动画与分数计算
5. 方块下落补位逻辑

## 关键技术
- Unity UI布局（Grid Layout Group）
- 碰撞检测与拖拽事件
- 简单动画（Scale缩放、Alpha渐变）

## 待优化点
1. 增加特殊方块（如炸弹、横向消除）
2. 优化拖拽手感（增加吸附效果）
